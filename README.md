# PrincipalPath
Repository for Principal Path finding algorithms, see References for details

# Installation
```
git clone https://github.com/MaJac89/PrincipalPath.git
cd PrincipalPath
source sourceme.sh
```

# Examples
```
cd PrincipalPath/examples
python rkm_2D.py
```
